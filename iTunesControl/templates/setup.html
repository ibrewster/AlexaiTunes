Alexa iTunes Control Server Setup

iTunes Library

Enter the location of your iTunes library XML file. By default, this is ~/Music/iTunes/iTunes Library.xml. If this file does not exist in your iTunes library, you may need to enable sharing the iTunes library XML with other applications in iTunes->Prefrences->Advanced. You can also find the location of your media folder there, if you need it.

Alexa iTunes User ID

Your Alexa iTunes user ID is assigned by the Alexa iTunes skill the first time you try to use it. Check the Alexa app on your phone for a card containing this value after activating the iTunes Alexa skill. It will be a 8 character string. This ties this instance of the server to your Alexa.
